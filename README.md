# FleetioDemo

## Prerequisites

- Xcode 15.0 or later
- iOS 17.0 or later
- Fleetio API Key / Token

## Configure API Credentials

I have not included any API credentials so new ones will need to be added into the Config file.

`FleetioDemo/Configs/Debug.xcconfig`

```ENV
API_KEY = 
API_TOKEN = 
```
